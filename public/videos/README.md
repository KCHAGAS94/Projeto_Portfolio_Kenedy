# Vídeos do Portfólio (Operação React)

Coloque seus vídeos com estes nomes para a página carregar automaticamente:

- `entrada-loop.webm` (hero de entrada, 5s em loop)
- `oee-bento.webm`
- `stock-bento.webm`
- `planner-bento.webm`
- `deep-dive.webm`
- `oee-gallery.webm`
- `stock-gallery.webm`
- `planner-gallery.webm`
- `consultoria.webm`

Se preferir, você pode usar `.mp4`; nesse caso atualize os `src` em `src/App.jsx`.
